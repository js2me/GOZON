# GOZON - ненастоящий маркетплейс – миллионы товаров по выгодным ценам    


Этот проект некоммерческий пример работы MobX в режиме SSR + демонстрация работы MobX библиотек   


## Что под капотом ?   

- react
- @heroui
- typescript
- mobx
- mobx-view-model
- mobx-location-history
- mobx-route

## Как запустить ?  

1. Нужно установить NodeJS
2. Нужно установить pnpm \ npm
3. `pnpm i` или `npm install`   
4. `pnpm dev` или `pnpm start` или `npm run dev` etc
